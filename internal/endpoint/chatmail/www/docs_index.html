فهرست مستندات

مستندات {{.WebDomain | cleanDomain}} را در اینجا مدیریت کنید.

  • مستندات عمومی اطلاعات پایه برای تمام کاربران.
  • مستندات مدیر سیستم مدیریت پیشرفته و دستورات خط فرمان.
  • مستندات Admin API رابط برنامه‌نویسی مدیریتی — تمام endpointها و نحوه استفاده.
  • راهنمای شخصی‌سازی HTML چگونه ظاهر این سرور را شخصی‌سازی کنید.
  • مستندات پایگاه داده نحوه اتصال به PostgreSQL و MySQL.
  • 🐳 راه‌اندازی با داکر نحوه اجرای سرور با Docker و docker-compose.
  • 🌐 راه‌اندازی دامنه و TLS اتصال دامنه واقعی، تنظیم DNS و گواهینامه Let's Encrypt.
