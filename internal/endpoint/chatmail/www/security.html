امنیت و حریم خصوصی

کاربران عزیز توجه داشته باشید که دلتاچت می‌تواند هم برای ارتباط امن و رمزنگاری‌شدهو هم برای ارتباط غیر امن (ایمیل سنتی)استفاده شود.

برای حفظ امنیت کامل، حتما موارد زیر را رعایت کنید:

  • توقف استفاده از ایمیل کلاسیک:در تنظیمات دلتاچت، بخش Advanced، گزینه «ایمیل سنتی» را روی «نه»قرار دهید.
  • تشخیص پیام‌های ناامن: پیام‌هایی که با آیکون پاکت‌نامه یا آیکون خاکستری ظاهر می‌شوند، در واقع ناامن هستند. آیکون خاکستری نشان‌دهنده غیرفعال بودن رمزنگاری است.
  • شروع چت فقط با لینک:به هیچ عنوان مستقیما به ID (آدرس ایمیل) پیام ندهید. ارتباط باید حتما از طریق اشتراک‌گذاری لینک به فرمت https://i.delta.chat/#.. شروع شود. هرگونه ID خارج از این فرمت ناامن است.
  • تفاوت سرورها:سرورهای chatmail(مانند tiiz.ir) تبادل ایمیل رمزنشده را کلا رد می‌کنند. اما madmailدر نسخه‌های قدیمی ممکن است اجازه ارسال غیرامن را بدهد که توسط صاحب سرور یا دیتاسنتر قابل مشاهده است.

نکته برای مدیران:اگر از نسخه‌های قدیمی Madmail استفاده می‌کنید، حتما سرور خود را به‌روزرسانی کنید تا امنیت کاربران حفظ شود.
