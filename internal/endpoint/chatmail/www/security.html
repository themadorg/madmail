Security & Privacy

Dear users, please note that DeltaChat can be used for both secure, encrypted and insecure (traditional email) communication.

To maintain full security, make sure to follow these guidelines:

  • Stop using classic email: In DeltaChat settings, under Advanced, set "Classic email" to "No".
  • Identifying unsafe messages: Messages that appear with an envelope icon or grey icon are insecure.
  • Start chats only via link: Never send messages directly to an ID (email address).
  • Server differences: chatmail servers reject unencrypted email exchange entirely.

Note for administrators: If you are using older versions of Madmail, make sure to update your server to protect user security.
