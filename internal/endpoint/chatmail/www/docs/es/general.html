Documentación general

Bienvenido a la documentación general del servidor Madmail.

Introducción a Madmail

Madmail es una aplicación de correo basada en el proyecto Maddy. Funciona como un único binario Go y su objetivo es ofrecer una plataforma de comunicación segura y rápida.

Integración con Delta Chat

La característica principal de Madmail es que crea un servidor de chat cifrado de extremo a extremo. Puede usar este servicio con la aplicación Delta Chat y disfrutar de un mensajero moderno en una plataforma privada y descentralizada.

Registro simple e instantáneo (JIT)

Una de las funciones inteligentes de Madmail es el registro instantáneo o Just-In-Time (JIT). No necesita pasos largos de alta: en el primer inicio de sesión con credenciales nuevas la cuenta se crea sola si JIT está activado. También puede usar el botón de registro en la página principal para obtener rápidamente un código QR con su configuración.

Llamadas de voz y vídeo (TURN)

El servidor incluye un servicio TURN dedicado para mejorar la calidad de las llamadas en Delta Chat. Actúa como relé cuando dos usuarios no pueden conectarse en directo (P2P), de modo que la llamada pasa por el servidor con buena estabilidad.

Cuota de almacenamiento

Para repartir recursos de forma equitativa, el servidor aplica límites de almacenamiento. Puede ver su uso y su cuota en cualquier cliente de correo que soporte la extensión QUOTA.

Administración del servidor (Admin API)

Desde la versión v0.15.3, el servidor incorpora una API de control (Admin API) para gestionar el servidor con facilidad. La ruta de la API y el token de administrador son configurables en el archivo de configuración.

Para obtener el token por defecto, ejecute en el servidor maddy admin-token.

Además de esta API, existe el proyecto Web Admin para gestión gráfica. El código fuente está en github.com/themadorg/madmail-admin-web y se construye y publica en https://admin.madmail.chat/. Abra ese enlace, introduzca la dirección de su servidor y el token para administrarlo.

Recomendaciones de seguridad

Seguridad de los mensajes: el servicio es seguro por defecto. En Delta Chat, un icono gris junto a los mensajes indica que el cifrado está desactivado; no envíe datos sensibles en ese caso.

Actualizaciones: mantenga Delta Chat actualizado para beneficiarse de las últimas mejoras de seguridad.

Soporte

Para soporte, contacte con el administrador de su servidor.
