Руководство по настройке внешнего вида сервера (Custom HTML)

Вы можете изменить внешний вид всех веб-страниц вашего сервера без изменения исходного кода или перекомпиляции.

Шаг 1 — Извлечение оригинальных файлов

Сначала извлеките файлы по умолчанию в папку. Мы рекомендуем использовать /var/lib/maddy/www.

sudo maddy extract-www /var/lib/maddy/www

Шаг 2 — Установка прав доступа

Чтобы сервер мог читать файлы, измените владельца папки на пользователя maddy.

sudo chown -R maddy:maddy /var/lib/maddy/www

Шаг 3 — Включение в конфигурации сервера

Обновите файл конфигурации сервера (maddy.conf), чтобы использовать новую папку.

sudo sed -i '/chatmail.*{/a\ www_dir /var/lib/maddy/www' /etc/maddy/maddy.conf

Шаг 4 — Применение изменений

Наконец, перезапустите службу, чтобы изменения вступили в силу.

sudo systemctl restart maddy
Важно: После выполнения вышеуказанных шагов вы можете открывать файлы внутри /var/lib/maddy/www с помощью редактора (например, nano) и изменять их. Изменения вступают в силу немедленно (после обновления страницы).

Заметка о мгновенной регистрации (JIT)

Если вы настраиваете главную страницу (index.html), обратите внимание, что логика регистрации меняется в зависимости от статуса JIT:

  • Если JIT включен, учетные данные пользователя генерируются случайным образом в браузере.
  • Если JIT выключен (но регистрация открыта), браузер использует API /new для создания аккаунтов.

Мы рекомендуем изучить JavaScript-код функции generateAccount в файле по умолчанию.

Сброс к настройкам по умолчанию

Если вы хотите снова использовать встроенные файлы программы:

sudo sed -i '/www_dir/d' /etc/maddy/maddy.conf sudo systemctl restart maddy
