Общая документация

Добро пожаловать в общую документацию сервера Madmail.

Введение в Madmail

Madmail — это мощное почтовое приложение, построенное на базе проекта Maddy. Оно работает как один исполняемый файл на Go (Golang), и его основной целью является создание безопасной и быстрой платформы для общения.

Интеграция с мессенджером DeltaChat

Самая важная особенность Madmail заключается в том, что он создает для вас чат-сервер с сквозным шифрованием (End-to-End Encrypted). Вы можете использовать этот сервис с популярным приложением DeltaChat и наслаждаться современным мессенджером на полностью приватной и децентрализованной платформе.

Простая и мгновенная регистрация (JIT)

Одной из умных функций Madmail является поддержка мгновенной регистрации, или Just-In-Time Registration. Это означает, что вам не нужно проходить длительные этапы регистрации; при первой попытке входа с новыми учетными данными ваш аккаунт создается автоматически, если эта функция включена. Вы также можете использовать кнопку регистрации на главной странице, чтобы быстро получить QR-код с вашими настройками.

Голосовые и видеозвонки (TURN)

Одной из отличительных особенностей этого сервера является наличие выделенного TURN-сервиса для улучшения качества звонков в DeltaChat. Этот сервис действует как ретранслятор, так что если два пользователя не могут подключиться напрямую (P2P), звонок направляется через сервер с высокой стабильностью.

Квота хранилища

Для обеспечения справедливого распределения ресурсов сервер применяет ограничения на объем хранилища. Вы можете просмотреть свое использование и квоту через любой почтовый клиент, поддерживающий расширение QUOTA.

Управление сервером (Admin API)

Начиная с версии v0.15.3, в сервер встроен API управления (Admin API), который позволяет легко управлять сервером. Путь к Admin API и токен администратора настраиваются и могут быть заданы статически в файле конфигурации сервера.

Чтобы получить токен по умолчанию для аутентификации, выполните команду maddy admin-token на вашем сервере.

В дополнение к этому API, был разработан проект Web Admin для графического управления сервером. Исходный код доступен по адресу github.com/themadorg/madmail-admin-web и автоматически собирается и развертывается по адресу https://admin.madmail.chat/. Посетите эту ссылку и введите адрес вашего сервера и токен для удобного управления сервером.

Рекомендации по безопасности

Безопасность сообщений: Этот сервис очень безопасен по умолчанию. Чтобы гарантировать конфиденциальность общения в DeltaChat, обратите внимание, что наличие серой иконки рядом с сообщениями означает, что шифрование отключено; поэтому конфиденциальные сообщения не должны иметь этого индикатора.

Обновления: Чтобы поддерживать безопасность и пользоваться последними функциями, всегда обновляйте приложение DeltaChat до последней версии.

Поддержка
